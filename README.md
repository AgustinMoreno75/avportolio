# Augusto Valmont

Sitio editorial personal construido con Next.js 15, TypeScript, Tailwind CSS v4, shadcn/ui y Framer Motion.

Dominio de marca: https://augustovalmont.com

## Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS v4
- shadcn/ui
- Framer Motion
- Lucide React

## Desarrollo

```bash
npm run dev
```

Abrí http://localhost:3000 para ver el sitio.

## Contenido

Los artículos del journal viven en `src/content/journal/*.mdx`.

La configuración editorial y el dominio principal viven en `src/content/site.ts`.

## Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
```

## Deploy

El proyecto está listo para desplegarse en Vercel sin configuración adicional.
